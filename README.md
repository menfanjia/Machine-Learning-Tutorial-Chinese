# Machine-Learning-Tutorial-Chinese
### 专门面向中文用户的机器学习相关的学习资料大集合

任何学习教程的**不在数量多，而在质量精**，几个优秀的、经典的教程学完吃透，比天天下载收藏一堆花里胡哨的资源帖却很少看要来的更实在。作者在这里集合一批**优质的**、并且**对中文用户友好的**各类机器学习相关的学习资料，希望能对大家的学习工作起到一定的帮助。
